# Baby Tracker Dashboard — Prototype Screenshots

Reference screenshots for the Pierre Tracker dashboard UI. Used by the **baby-tracker-dashboard** OpenSpec change.

| File | Description |
|------|--------------|
| `01-dashboard-todays-summary.png` | Desktop: header ("Pierre Tracker" + date), Today's Summary 2×2 grid (Last Feed highlighted, Fed Today, Slept Today, Diapers Today), start of Feeding Volume chart |
| `02-feeding-chart-recent-activity.png` | Feeding Volume (7 Days) bar chart with "Mar DD" labels; Recent Activity section heading |
| `03-recent-activity-timeline.png` | Recent Activity event list: Feed, Diaper, Sleep rows with icons, details, timestamps; "View All History" link at bottom |
| `04-recent-activity-with-dates.png` | Same timeline with date selector (Mar 07–Mar 12) above Recent Activity |
| `05-todays-summary-mobile.png` | Mobile: Today's Summary in single-column stack (Last Feed on top, then Fed Today, Slept Today, Diapers Today) |

## Layout notes

- **Desktop**: Today's Summary uses a 2×2 grid; Last Feed card is blue and prominent (top-left).
- **Mobile**: Today's Summary stacks vertically; same four cards, full-width.
- Chart section title: "Feeding Volume (7 Days)". Timeline section title: "Recent Activity".
